## 第二部分：深度学习基础

* 回归问题
由二元一次方程组引入，介绍怎么处理连续值预测问题，包含误差的构建、计算公式的构建、误差的优化等；并通过一个简单的回归实战让读者对连续值预测问题有一个初步的印象。

* 分类问题
介绍完连续值预测，进而介绍离散值预测。首先解决分类问题的误差构建，前向传播的构建；最后通过一个MNIST小实战介绍给读者深度学习的第一印象，为接下来介绍TensorFlow铺垫。
